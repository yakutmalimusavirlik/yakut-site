# Yakut Mali Müşavirlik Website

## v1.2 — Design System + Structured Build

Bu sürümde:
- Tek dosya yapısından profesyonel klasör yapısına geçildi.
- CSS `assets/css/style.css` içine alındı.
- JS için `assets/js/script.js` oluşturuldu.
- Border vurguları güçlendirildi.
- Yakut kırmızısı sadece vurgu noktalarında kullanıldı.
- Bilgilendirme Merkezi ve Makaleler bölümleri "Yakında Yayında" durumuna alındı.
